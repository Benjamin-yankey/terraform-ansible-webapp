# Deployment Evidence

This directory contains deployment evidence and verification files.

## Files (Generated during deployment)

- `APPLY.txt` - Terraform apply output
- `DESTROY.txt` - Terraform destroy output
- `deployment_vars.json` - Deployment variables
- `public_ip.txt` - EC2 instance public IP
- `terraform_outputs.txt` - Terraform outputs

## Note

These files are generated automatically and should not be committed to version control.
